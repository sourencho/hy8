# hy8

An armenian font for [PICO8](https://www.lexaloffle.com/pico-8.php).

Original font [hypix](https://fonter.am/en/fonts/hypix) by Davit Saroyan.

![](images/hy8.png)

## use

See [hy8_demo.p8](demo.p8)

## modify

See [hy8.p8](hy8.p8)
